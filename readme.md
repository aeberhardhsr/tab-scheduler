# Tab scheduler

## Information
Das Script steuert die zeitliche Abfolge der festgelegten Tabs in Google Chrome. 

## Installation
Script öffnen und Parameter anpassen.

## Gebrauch
Script in Kommandozeile mit `python scheduler.py` ausführen.
